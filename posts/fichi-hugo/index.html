Иконка со ссылкой

Параметры:

  • vendor: Поставщик
  • link: Ссылка
  • name: Название иконки
{{< icon vendor="" name="" link="" >}}

Доступны иконки из: Feather icons

имя: svg
globe
moon
sun
github
facebook
rss
twitter
mail
